# Agriculture-management-system
Agriculture Management System built using Java Swing and MySQL. It allows users to manage crops with CRUD operations, track progress, generate reports, and visualize data using charts.
A Java-based desktop application developed using Java Swing and MySQL to manage agricultural data efficiently. This system allows users to perform CRUD operations on crops, track their growth progress, and generate reports with a user-friendly interface.

🚀 Features
🔐 User Login & Registration
🌱 Crop Management (Add, View, Update, Delete)
🔍 Search and Filter Crops
📊 Data Visualization using Charts
📄 PDF Report Generation
📈 Progress Tracking of Crops
🎨 Modern GUI using Java Swing
🛠️ Technologies Used
Frontend: Java Swing
Backend: Java (JDBC)
Database: MySQL
🗄️ Database Structure
Users Table: Stores user login details
Crops Table: Stores crop information
Progress Table: Tracks crop growth stages
▶️ How to Run
Clone the repository
Import project into VS Code
Add MySQL Connector (JDBC driver)
Create database agriculture_db in MySQL
Update DB credentials in DBConnection.java
Compile and run:
javac *.java
java MainUI
📌 Project Objective

To provide a simple and efficient system for managing agricultural data digitally, improving productivity and data organization.
